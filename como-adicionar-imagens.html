Como adicionar imagens aos imóveis?

  1. Para que todos vejam a imagem do seu imóvel, ela precisa estar hospedada na internet.
  2. Recomendamos usar o Firebase Storage para hospedar suas imagens.
  3. No formulário de anúncio, selecione a imagem do seu computador ou celular.
  4. Ao clicar em "Anunciar", a imagem será enviada para o Firebase Storage e ficará disponível para todos.
  5. Se você não usar o Firebase Storage, a imagem não aparecerá para outros usuários.

Como configurar o Firebase Storage?

  1. Acesse o console do Firebase e selecione seu projeto.
  2. No menu esquerdo, clique em Storage e depois em Criar Storage.
  3. Siga os passos para ativar o Storage.
  4. Nas regras do Storage, para testes, use:
    allow read, write: if true;
  5. Salve as regras e pronto! Agora seu site pode hospedar imagens.

Se precisar de ajuda, entre em contato pelo menu "Contato".
